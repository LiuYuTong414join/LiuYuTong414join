  # 工会系统设计
  ## 工会的创建与加入
  ### 创造条件
  - 到达一定等级
  - 支付一定游戏货币
  
  ## 加入方式
  - 可以自己主动加入
  - 可以由工会成员邀请进入
  - 退出公会后24小时内不可加入工会
  
  ## 职位
  - 会长一名
  - 副会长一名
  - 左右护法各一名（左护法仅限男体 右护法仅限女体）
  - 长老初始一名，后续帮会升级后可增员至两名
  - 精英初始两名，后续帮会升级后可增员至六名
  - 普通成员若干
  > 加入工会后即可获得称号，不同称号有不同的属性加成

  ## 工会功能
  - 聊天系统：工会有自己的专属聊天频道，成员可以在此交流
    >会长 副会长有禁言权限
    >会长 副会长 护法 长老可开启帮会语音

 - 工会任务：每日凌晨五点更新每日任务，成员可完成任务获得工会贡献，提升工会等级
 - 工会副本：每周一凌晨五点更新工会副本，只能由本工会成员组队完成，会长可设置难度，不同难度奖励不同
 - 工会技能：成员可使用工会贡献学习并升级工会技能提升自身属性
 - 工会商店：每周一凌晨五点刷新，使用公会贡献可在商店中兑换奖励
 - 工会切磋: 每周五晚七点根据工会排名匹配帮会进行帮会切磋，不同的两个帮会进行切磋，每个帮会限制50人参与
 - 工会排行榜：根据帮会贡献排，榜上前十的工会有奖励
  
  ## 工会升级
  ### 条件
  - 达到一定人数
  - 消耗一定帮会贡献值
  - 会长可开启
  ### 奖励
  - 增员
  - 发放奖励
  - 解锁更多工会技能
  - 扩大工会商店规模
  
  ## 工会权限
  - 会长 副会长可将普通成员踢出工会，赋予普通成员职位
  
  # APP设计
  ## 产品名称：萌宠宝

  ## 产品定位：
  萌宠宝是一款面向宠物爱好者的休闲云养宠APP。养宠爱好者们可通过此APP学习更多养宠知识，分享萌宠日常，购买宠物用品，宠物服务预约，云养电子宠物。
  ## 市场背景
  - 宠物市场需求
  - 科普动物相关知识
  ## 业务需求
  1. 养宠攻略
   - 用户可在此APP上查看不同动物的饲养攻略
   - 支持宠物主人们分享交流自己的养宠经验
  2.宠物用品购买
   - 根据不同用户需求推荐不同的宠物用品，支持在线购买
   - 提供丰富的宠物食品，用品，玩具等
  3.宠物服务预约
   - 提供宠物美容、洗澡、寄养、训练等服务的预约功能
   - 支持在线支付和评价，方便宠物主人选择优质的服务
   - 提供服务商家的信息和评价，帮助宠物主人选择可靠的服务商家
  1. 云养宠
   - 用户可养自己的电子宠物
  2. 宠物社交
   - 提供服务商家的信息和评价，帮助宠物主人选择可靠的服务商家。
   - 举办宠物相关的活动和比赛，增加宠物主人的参与度。
  ## 具体功能实现的意义
  -  宠物社交功能：让宠物主人可以分享宠物的生活点滴，增加宠物主人之间的交流和互动，丰富宠物主人的生活。
  - 宠物用品购物功能：为宠物主人提供便捷的购物渠道，让宠物主人可以轻松购买到所需的宠物食品、用品等商品。
  - 宠物服务预约功能：方便宠物主人预约宠物美容、洗澡、寄养等服务，提高宠物服务的效率和质量。
  - 养宠攻略功能：帮助宠物主人提高养宠水平，更好地照顾宠物，减少宠物疾病的发生。
  - 云养宠功能：帮助无法现实生活中养宠的用户实现养宠梦。
  ## 实现该功能的技术
  - 人工智能技术：利用人工智能技术，为宠物主人提供个性化的养护建议和服务推荐。 生成不同的电子宠物。
  -  移动支付技术：在宠物用品购物和服务预约功能中，利用移动支付技术，实现安全、便捷的在线支付。
  -  社交媒体技术：在宠物社交功能中，利用社交网络技术，实现用户之间的交流和互动。
